Seattle之行的杂七杂八

By xym

2017-05-28 更新日期:2017-05-30

　　从西雅图回来快半个月，才没多久我就开始憧憬着下一次要去哪，看什么风景，体验什么样的生活，果然我骨子里还是躁动不安的。翻了翻我那零碎的微博，想着确实还没写完这段旅程的经历，于是补上。

　　Focus，2017年初自己给今年定的基调是集中在学术上，其他的事情都可以放下。这几个月我也是这样过来的，一周基本上80%的时间都泡在实验室，paper也陆陆续续写了几篇，且不说质量如何，单纯就写作经验来讲感觉进步挺大，对论文结构、语言的把握都变的熟练了。可是长时间的实验室生活让我的情绪波动太大，怎么说呢？因为效率实在不高，加上太过单一的节奏，让自己变的敏感起来，非常容易被生活中的小事所影响。比如周末吃饭的问题，运动的问题，喝咖啡的问题，天气的问题。来西雅图的这段时间，基本上没怎么做学术，开会的时间开会，闲逛的时候闲逛。我觉得自己特别享受这种生活，无论是在会场上听报告，尝试与大牛们沟通问题，还是在这座城市里面行走，拍照，生活。愉悦的情绪反馈给我非常棒的体验，使得我也很focus于我这几天的生活。所以我有在思考一个问题，什么样的程度才算的上focus，什么样的生活才算的上focus。

　　在西雅图的GasworksPark，遇到了一个喜欢摄影的俄罗斯阿姨。她为了拍摄日落和月升在湖边持续了5个小时。我在跟她聊天的时候，她说她photograph的意义是享受这个过程，所以她非常喜欢拍摄风景，喜欢发现探索的这个经历。第一反应，我想到的是三年前的自己。为了捕捉早晨和傍晚的光线、拍摄午夜的星空，可以废寝忘食。可是仔细想想，当时的自己有在享受这个过程吗，还是仅仅沉醉在器材上或者是别人的称赞中？而且现在的自己花在拍摄上的时间愈发的少了。她打开她的Flickr给我看了她去过的地方，拍过的风景，跟我分享她拍摄时遇到过的趣事。我觉得自己很羞愧，并不是因为我们之间拍摄的水平差距，而是在于对待这件事情上的投入程度。都是喜欢，我却学会了妥协，向自己的懒惰和安逸妥协，向别人的需求妥协。回忆了自己二十几年的人生，好像再也找不出一件自己引以为傲同时也称得上focus的事了。（有自己的原因也有家庭的原因，这个我整理整理在另外一篇文里面聊聊吧，关于童年生活，关于对未来孩子的童年生活憧憬的一些话）

　　在我这个年龄，已经可以主导自己的生活了。所以我觉得要做的事一定要是自己喜欢的事，愿意付出全身心去做的事，面对困难和阻碍也不会放弃的事。愉悦了自己，才会带来继续下去的勇气和动力，才会做的更好，这是最简单不过的正反馈了，这样才能focus在这件事上。所谓顺心意也是如此吧。顺心意，顺的是自己的心意，如果都不能说服和取悦自己，那自己做这件事又有什么意义呢？

　　再说一点点其他的感触。每到一个新的地方旅行，我会去体验当地人的生活。吃早餐，去教堂，看电影，大笑和交流，在新环境下自己变得更加的开阔起来。我喜欢看到这样的自己，有所求，不拘束。突然想到几句歌词，可以很恰当的描述我的这份感受，“我想我的你，是自由的，是开阔的，却能惦着我；我想各自经历许多，风雨喜乐，有故事可以说。”　祝愿自己和未知的那个人，能够过着精彩的人生，将来可以分享很多很多美好的故事。
