Start of Something New

2016-06-07 • 分类: -eassy

2016’s plan

　　首先是今年年初在新浪微博写的2016计划。

　　学术：当然就是开题。开题需要有文章啊，所以目标就是1期刊加1会议吧，开题当然也得过。

　　身体：继续跑步游泳是必须的，体重维持在140左右，把多余的脂肪练掉。还有就是改正驼背，保持一个良好的体形。

　　摄影：手机继续保持下去，等365结束以后改成每月总结一次吧，毕竟今年重心不在这个上面。夏初找个机会再拍次银河拱桥。有时间还是多爬爬楼。剩下的就是拍女友咯。跟随大师们的脚步。

　　书和电影：2015看的不少，写下的感受太少。所以继续看的同时，可以多写写，不要求一定成篇，短短几句话也可以。

　　旅行：想跟老爸租车自驾一次，暂时考虑的西藏的阿里环线（有点远加不靠谱）。跟义哥醒哥自驾一次北京周边。跟女友一次日本或者台湾。

　　沟通：处理好与老爸现在略病态的关系。增加对老妈的问候和关心。重新联系上一些很久没见的同学。利用假期去看下几位老师。真心对待现在的朋友，揶揄人的话一定要减少说。还有就是认真对待跟女友可能出现的问题和矛盾，面对面交流，吵架不过夜。

　　生活：规律作息。抓住琐碎的时间，做有意义的事。珍惜大块的时间，发挥应有的效果。练字还是继续做。减少出去吃的频率，一个月最多5次。花钱之前先考虑清楚重要程度和能否替代的可能性。

2016/6 New Start

　　其次是半年来完成的情况，以及最近的一些感想和开这个blog的初衷。

　　学术：开题被拖到了下学期。急需加强的是与师兄师弟讨论问题的程度，集思广益。

　　身体：健身房ing,不过频率需要加强。长时间没有游泳了，应该重新拾起。驼背纠正中，有一些效果。

　　摄影：已经沦为纯粹的手机摄影户…

　　书和电影：说好的评论呢？

　　旅行：去了一趟天津。暑假回家带父母自驾广西，从北到南。

　　沟通：努力中…

　　生活：重新买了基金。时间方面急需加强的是 不要花时间做没有意义的事！！！！！

　　之前想到开blog只是为了找个地方总结一些技术资料，辗转看了一些博客，太过凌乱，不如自己搭个简洁的，于是就有了现在的模样。

　　最后自勉一下，与其醉生梦死，不如负荆前行。
