ROA边缘检测算子

2016-06-16 • 分类: -study • 标签: -SAR -image processing -edge detection

ROA

　　图像梯度可以理解为计算两个正交方向邻域像素平均值的变化的均方差。
${\rm{|R}}(x,y) - L(x,y)|$是$(x,y)$邻域的变化平均值。对其使用对数值，可以得到：
$${\rm{|logR}} - \log L| = \max (\log (R/L),\log (I/R))$$
所以对于具有乘性噪声的SAR图像，邻域像素的变化通常用两者的比值来表示，而不是加性噪声图像中的差。
$$H(x,y) = \max(R/L,L/R)$$
　　所以ROA梯度算子，Ratio of average，常用的一个估值表达式为：
$$RoA(x,y) = \sqrt{H^{2}(x,y) + V^{2}(x,y)}$$
　　ROA存在的问题：选取的窗口太小，例如3X3，可以比较准确的提取出边缘，但是因为对局部灰度值的变化很敏感，会得到很多的虚假边缘；选取的窗口变大，提取出的边缘定位性能减弱，且得到的边缘变厚，但对局部变化鲁棒，因为可以理解成先做了一个均匀滤波。

ROA的各种改进

LOG-ROA

　　先做LOG，求出过零点，然后对过零点应用ROA算子。LOG过零点得出的边缘厚度总是一个像素，但是不包含梯度幅度信息；ROA算子得出的边缘总是很厚，梯度幅度被用于阈值。

MRoA+RGoA

　　MRoA:求取四个方向的邻域平均值的比值，竖直、水平、45°、135°
$${R_i} = \min ({P_i}/{Q_i},{Q_i}/{P_i});i = 1,2,3,4$$
$$R = \min({R_1},{R_2},{R_3},{R_4})$$
　　RGoA:求取四个方向的邻域平均值的差值
$${G_i} = |{P_i} - {Q_i}|;i = 1,2,3,4$$
$$G = \max({G_1},{G_2},{G_3},{G_4})$$

MSP-ROA

　　对于每个点，计算梯度幅度R和方向O，R通过MRoA的方法计算，四个方向中最小值的方向被记为O。在方向O上该点的两侧各选取d-1个点组成sub-window，D=2d-1.如果在sub-window中，候选点的值是最小的，则保留为边缘点，否则去除。

ROEWA

　　考虑到一个多边缘的图像模型，为了计算局部灰度均值，设计了一个线性最小均方误差滤波器，得到了指数加权的均值计算方式。指数加权的求均值方法只对统计图像模型的场景是严格适用的。
$${f_1}(n) = a \cdot {b^n}u(n);{f_2}(n) = a \cdot {b^{ - n}}u( - n)$$
$$f(n) = c \cdot {b^{|n|}} \equiv \frac{1}NaN{f_1}(n) + \frac{b}NaN{f_2}(n)$$
$${\mu_1}(x,y) = {f_1}(x)\ast (f(y)\ast I(x,y))$$
$${\mu_2}(x,y) = {f_2}(x)\ast (f(y)\ast I(x,y))$$

Modified ROEWA

　　在ROEWA的基础上加入了边缘方向，对梯度幅度图点乘不同方向的Gabor滤波器，通过判断出最大的值，将Gabor滤波器的方向当作边缘方向。

ROA+Hough

　　先通过Hough变换得到包含边缘的区域，对不同的区域使用ROA算子有后采用不同的阈值T。对于强对比区域（计算变化系数，变化系数小的），直接应用T；对于弱对比区域（变化系数大的）应用T+0.1；对于没有明显边缘的区域，应用T-0.1。

ROA+GGSwindow

　　使用矩阵窗口时，在真正边缘的附近总是会引起一些虚假的边缘。矩阵窗口不是好的平滑滤波器，强的斑点噪声使得按照矩阵窗口计算局部均值的函数包含了一些不想要的高频分量。

水平方向的窗口对如下所示：
$${W_U}(x,y) = \frac{|y{|^{\alpha - 1}}}{\sqrt {2\pi }{\sigma_x}\Gamma (\alpha ){\beta ^\alpha }}\exp ( - (\frac{x^2}{2\sigma_x^2}+\frac{|y|}{\beta})),y \ge 0$$
$${W_U}(x,y) = \frac{|y{|^{\alpha - 1}}}{\sqrt {2\pi }{\sigma_x}\Gamma (\alpha ){\beta ^\alpha }}\exp ( - (\frac{x^2}{2\sigma_x^2}+\frac{|y|}{\beta})),y \le 0$$
水平方向上是Gauss函数，竖直方向上是Gamma函数。
方向为$\theta$的窗口对：
$$W_U^\theta (x,y) = {W_U}(x\cos \theta - y\sin \theta ,x\sin \theta + y\cos \theta )$$
$$W_L^\theta (x,y) = {W_L}(x\cos \theta - y\sin \theta ,x\sin \theta + y\cos \theta )$$
局部均值如下计算：
$$m(x,y|\theta) = \sum_i {W^\theta}({x_i},{y_i})I(x - {x_i},y - {y_i})$$
